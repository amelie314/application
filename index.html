5. 請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

在自學的過程中，偶爾會感到焦慮，尤其是當進度落後於預期時。由於缺乏實體同學間的互動和交流，也擔心自己是否能夠有效地學習和掌握新的技術。 為了處理這些負面情緒，我嘗試採取以下行動：

  •  與朋友建立連結，通過線上討論(例如: slack)來增進互動和交流。
  •  使用生產力工具輔助，例如分配學習時間和集中注意力，減輕焦慮感。

  • 透過這些方法處理負面情緒，保持學習的動能。

6. 關於這份申請網頁，請分享一個你開發時的技術心得。

此申請網頁我使用了HTML和CSS來實現網頁的結構和外觀。過程中：

  • 選擇了合適的 HTML 標籤和 CSS 樣式來組織網頁內容並設計網頁的外觀和版面配置。我根據申請網頁的需求和目標，選擇了適合的技術並應用於實際開發中。例如，我使用了語意化的 HTML 標籤來描述網頁的內容結構，並使用 CSS 樣式來控制網頁的外觀，包括字體、顏色、佈局等。這有助於提高網頁的可讀性和維護性，使得網頁結構和樣式更加清晰和易於管理。
  • 其次，我運用了一些基本的設計模式和架構來組織和管理網頁的結構和樣式。我遵循了「分離關注點」的原則，將 HTML 和 CSS 進行分離，使得網頁結構和樣式能夠更容易地維護和擴展。我使用了 CSS 的類和 ID 選擇器來達到樣式的重用性和可讀性，並使用了層疊樣式表（CSS）來控制樣式的層次關係和繼承。這些設計模式和架構有助於使得網頁的開發過程更加組織化和高效。
